```bash

# 输出包含:
# - 数据概览（行数、列数、时间范围）

# - 洞察发现（趋势、异常、相关性）
```

#### 5. 导出报告

```bash
# 导出完整分析报告
datamind export sales_report.md --title "销售数据分析报告"
```

## 多表关联查询

### 场景描述

当有多个数据表时，DataMind 会自动选择相关表进行分析。

### 数据准备

```bash
# 创建订单表
cat > orders.csv << 'EOF'
order_id,customer_id,product_id,quantity,order_date,total_amount
O001,C001,P001,2,2024-03-15,5998
O002,C002,P002,1,2024-03-15,7999
O003,C003,P001,1,2024-03-16,2999
O004,C001,P003,3,2024-03-17,8997
O005,C004,P002,2,2024-03-18,15998
EOF

# 创建客户表
cat > customers.csv << 'EOF'
customer_id,name,city,level,join_date
C001,张三,北京,金卡,2023-01-15
C002,李四,上海,银卡,2023-03-20
C003,王五,广州,普通,2023-06-10
C004,赵六,北京,金卡,2023-02-28
EOF

# 导入数据
datamind import orders.csv
datamind import customers.csv
```

### 分析示例

#### 1. 查看所有表

```bash
# 列出所有表
datamind list

# 输出:
# ┌───────────┬────────┬───────────────────────────────────────┐
# │ 表名      │ 行数   │ 列                                    │
# ├───────────┼────────┼───────────────────────────────────────┤
# │ orders    │ 5      │ order_id, customer_id, product_id, ... │
# │ customers │ 4      │ customer_id, name, city, level, ...    │
# └───────────┴────────┴───────────────────────────────────────┘
```

#### 2. 指定表查询

```bash
# 查询订单表
datamind ask "订单金额最高的客户" -t orders

# 查询客户表
datamind ask "北京的金卡客户有哪些" -t customers
```

#### 3. 自动表推荐

```bash
# DataMind 会根据问题内容自动推荐相关表
datamind ask "客户的订单数量"

# 问题中包含"客户"，会优先查询 customers 表
datamind ask "各城市的客户数量"
```


```bash
# ✅ 推荐
```

## 常见问题

### Q1: 查询结果不准确？

确保问题表述清晰，包含必要的限定词和度量。

### Q2: 图表显示不正确？

检查数据类型是否适合选择的图表类型：
- 柱状图：适合分类对比
- 折线图：适合时序数据
- 饼图：适合占比分析
- 散点图：适合相关性分析

### Q3: 多表查询时结果不对？

使用 `-t` 参数明确指定要查询的表。

### Q4: 如何处理大数据量？

DataMind 默认显示前 50 行结果。如需完整数据，可导出为文件：

```bash
datamind export result.csv
```
