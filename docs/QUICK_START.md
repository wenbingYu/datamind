# 5 分钟快速入门

本指南帮助你在 5 分钟内快速上手 DataMind，体验从数据导入到智能分析的完整流程。

## 前置条件

确保已完成以下步骤：

- ✅ 安装 Node.js 18+
- ✅ 安装 DataMind (`npm install -g datamind`)
- ✅ 配置 API Key (`export DATAMIND_API_KEY=your_key`)

> 📖 未完成安装？请参考 [安装指南](./INSTALLATION.md)

---

## 第一步：设置 API Key

DataMind 需要 LLM API Key 来支持自然语言查询功能。

```bash
# macOS / Linux
export DATAMIND_API_KEY=your_api_key_here

# Windows PowerShell
$env:DATAMIND_API_KEY="your_api_key_here"
```

> 💡 **提示**: 你可以使用阿里云百炼、智谱 AI 或 OpenAI 的 API Key。推荐永久设置到 shell 配置文件中。

---

## 第二步：导入第一个数据集

让我们创建一个销售数据示例并导入：

```bash
# 创建示例 CSV 文件
cat > sales.csv << 'EOF'
date,product,region,quantity,amount
2024-01-15,iPhone 15,北京,10,79990
2024-01-16,MacBook Air,上海,5,54995
2024-01-17,iPad Pro,广州,8,47992
2024-01-18,AirPods Pro,深圳,20,13980
2024-01-19,iPhone 15,上海,15,119985
2024-01-20,MacBook Pro,北京,3,53997
2024-01-21,iPad Air,杭州,12,35988
2024-01-22,Apple Watch,成都,18,8982
2024-01-23,iPhone 15 Pro,北京,8,95992
2024-01-24,MacBook Air,深圳,6,65994
EOF

# 导入数据
datamind import sales.csv
```

**预期输出**：

```
✔ 导入成功!

  表名: sales
  行数: 10
  列数: 5

使用 'datamind ask "问题"' 查询数据
```

## 第三步：执行第一个查询

用自然语言查询数据，无需编写 SQL：

```bash
# 查询销售额最高的产品
datamind ask "销售额最高的产品是哪个？"
```

**预期输出**：

```

│ iPhone 15 Pro  │ 95992      │
└────────────────┴────────────┘
```

## 进阶使用

### 生成图表

```bash
# 生成柱状图
datamind ask "各产品的销售额" -c bar

# 生成折线图
datamind ask "销售额趋势" -c line

# 生成饼图
datamind ask "各地区销售占比" -c pie

# 保存图表到文件
datamind ask "产品销售排行" -c bar -o chart.html
```

### 导出报告

```bash
# 导出 Markdown 报告
datamind export report.md

# 导出 HTML 报告
datamind export report.html --title "销售分析报告"

# 导出 JSON 数据
datamind export data.json
```

### 查看数据表

```bash
# 列出所有数据表
datamind list

# 预期输出:
# ┌──────────┬────────┬──────────────────────────────────────┐
# │ 表名     │ 行数   │ 列                                   │
# ├──────────┼────────┼──────────────────────────────────────┤
# │ sales    │ 10     │ date, product, region, quantity, ... │
# └──────────┴────────┴──────────────────────────────────────┘
```

## 查询技巧

### 明确指定表名

当有多个数据表时，指定要查询的表：

```bash
datamind ask "销售额最高的产品" -t sales
```

### 使用中文提问

DataMind 支持中文自然语言查询：

```bash
# ✅ 推荐：清晰的中文问题
datamind ask "上个月销售额超过10万的产品有哪些？"
datamind ask "各地区销售数量和金额的汇总统计"
datamind ask "找出销售额最高的前三名产品"

# ❌ 避免：模糊不清的问题
datamind ask "数据怎么样"  # 太模糊
datamind ask "查一下"      # 没有具体目标
```

### 查询模式参考

| 查询类型 | 示例问题 |
|---------|---------|
| 排序 | "销售额最高的10个产品" |
| 汇总 | "各地区销售总额" |
| 筛选 | "北京地区1月份的销售记录" |
| 统计 | "平均单价、最高单价、最低单价" |
| 对比 | "iPhone 和 MacBook 的销量对比" |
| 趋势 | "销售额月度变化趋势" |
